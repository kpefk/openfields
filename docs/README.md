# OpenFields Documentation

Developer documentation lives here. Sections are added as the corresponding
features land:

- **Getting Started** — installation and local development.
- **Public PHP API** — `get_field()`, `get_fields()`, `update_field()`,
  `have_rows()` / `the_row()`, and the registration helpers.
- **Field Type API** — building custom field types (PHP + TypeScript contract).
- **Hooks Reference** — `do_action` / `apply_filters` extension points.
- **REST API** — endpoints and versioning policy.

See [`../CONTRIBUTING.md`](../CONTRIBUTING.md) for coding standards.
